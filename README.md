# Consulta Fácil

Site simples desenvolvido como parte de um trabalho acadêmico de Letramento Digital.
O objetivo é ajudar pessoas idosas a encontrarem materiais de apoio para marcar consultas em convênios de saúde.

## Site publicado

[Acessar o Consulta Fácil](https://marqueconsultafacil.netlify.app/)

## Materiais disponíveis

O site apresenta folhetos em PDF e vídeos explicativos para:

- Prevent Senior
- Santa Casa de Bragança
- Notredame

## Apresentação

Os slides do projeto estão disponíveis em PDF:

[Abrir apresentação do livreto](./Apresenta%C3%A7%C3%A3o%20livreto.pdf)

## Estrutura do projeto

- `index.html`: página principal do site.
- `hero-casal.png`: imagem usada na página inicial.
- `prevent-senior2.pdf`: folheto da Prevent Senior.
- `santa-casa-bragaca3.pdf`: folheto da Santa Casa de Bragança.
- `notredame2.pdf`: folheto da Notredame.
- `Apresentação livreto.pdf`: apresentação em slides do projeto.
- `_headers`: configurações de segurança usadas no deploy da Netlify.

## Observação

Este projeto é um site estático, sem banco de dados, login, formulário ou coleta de informações pessoais.
